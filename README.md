# 🚗 Austo Analytics Project

## 📌 Overview
This project focuses on analyzing customer data from an automobile company (**Austo Motor Company**) to improve the effectiveness of marketing campaigns using data-driven insights. The analysis helps identify customer behavior patterns and supports strategic decision-making.

---

## 🎯 Objective
- Analyze customer dataset to understand purchasing behavior  
- Identify key factors influencing vehicle selection  
- Validate business assumptions using data  
- Provide actionable insights for targeted marketing  

---

## 📊 Dataset Information
- **Total Records:** 1581  
- **Total Features:** 14  
- **Data Types:** Integer, Float, Object  

### Includes:
- Demographics (Age, Gender, Marital Status)  
- Financial Data (Salary, Partner Salary, Total Salary)  
- Customer Behavior (Loans, Dependents, Profession)  
- Product Info (Vehicle Type: SUV, Sedan, Hatchback)  

---

## 🛠️ Work Performed

### 🔹 Data Cleaning & Preprocessing
- Handled missing values in *Gender* and *Partner Salary*  
- Fixed inconsistent entries (e.g., "Femle", "Female")  
- Ensured data consistency  

### 🔹 Exploratory Data Analysis (EDA)
- Performed **univariate analysis** using charts  
- Conducted **bivariate analysis**  
- Used visualizations to identify patterns  

### 🔹 Key Analysis Areas
- Customer distribution (gender, profession, education)  
- Income & purchasing power  
- Vehicle preferences (SUV, Sedan, Hatchback)  
- Impact of loans & partner income  

---

## 📈 Key Insights
- Sedans are the most preferred vehicle  
- Salaried individuals dominate purchases  
- Age group 20–30 is most active  
- Female customers show higher SUV preference  
- Working partner → higher purchasing capacity  

---

## 🧠 Business Recommendations
- Target **salaried professionals** for Sedan campaigns  
- Focus **SUV marketing on female customers**  
- Create **family-based offers** for married buyers  
- Target **mid-income segment (₹50k–₹100k)**  
- Use **segmented marketing strategies**

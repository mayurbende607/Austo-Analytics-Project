📊 Austo Analytics Project
🔍 Overview

This project focuses on analyzing customer data from an automobile company to improve marketing strategies using data-driven insights.

📁 Dataset
1581 rows, 14 features
Includes demographic, financial, and purchase behavior data
⚙️ Work Done
Data Cleaning (handled missing values & inconsistencies)
Exploratory Data Analysis (EDA)
Univariate & Bivariate Analysis
Customer Segmentation
Business Insight Generation
📈 Key Insights
Sedan is the most preferred vehicle type
Salaried individuals dominate purchases
Female customers show higher SUV preference
Working partner → higher purchasing capacity
🧠 Business Impact
Helped design targeted marketing strategies
Identified high-value customer segments
Improved campaign decision-making
🛠 Tech Stack
Python
Pandas, NumPy
Matplotlib, Seaborn
